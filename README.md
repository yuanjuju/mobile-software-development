# 移动软件开发 — 课程实验合集

## 项目简介

本仓库为 2024 夏移动软件开发课程的实验分享仓库，收录了多个移动端开发实践项目，涵盖 Android 及跨平台开发相关内容。

## 内容结构

```
├── 实验一/    # 基础 UI 开发
├── 实验二/    # 交互与事件处理
├── 实验三/    # 数据存储与网络请求
├── 实验四/    # 多媒体与传感器
├── 实验五/    # 综合应用开发
└── 实验六/    # 项目优化与发布
```

每个实验文件夹中均包含实验指导书（`.md` / `.pdf`），可自行查看。

## 使用说明

1. 按文件夹名查看对应实验
2. 每个实验包含完整的项目代码
3. 仅供参考学习，严禁抄袭
